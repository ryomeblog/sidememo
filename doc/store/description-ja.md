# ストア掲載文（日本語）

## 短い説明（132 文字以内 / Chrome ウェブストア「概要」用）

ブラウザ右側のサイドパネルで使える Markdown メモ。リアルタイム整形、タグ・全文検索、ページ URL の添付、JSON でのバックアップに対応。データはすべてローカル保存で外部送信しません。

## 詳細説明（4,000 文字以内）

SideMemo は、ブラウザのサイドパネルで動作する Markdown 対応のメモ取り拡張機能です。調べ物の最中でも、ブログや学習メモを書きたいときでも、タブを離れずにそのまま書き始められます。

### 特長

- **リアルタイム整形の Markdown エディタ**
  入力したそばから `#` が見出しに、`**` が太字に、`- [ ]` がチェックボックスに整形されます（編集モードとプレビューモードの切替はありません）。

- **3 種類のレイアウトモード**
  自動 / 常時 2 ペイン / 常時 1 ペインから選択できます。自動モードはパネル幅 400px をしきい値に、メモ一覧とエディタの並列表示と、タブでの切替表示を切り替えます。

- **タグと全文検索**
  Inbox / Idea / Work の初期タグから始められます。タグは色付きで自由に追加・編集でき、複数選択での絞り込みにも対応。検索はタイトルと本文を横断するあいまい検索です。

- **ページ参照（PageRef）**
  「+ ページ添付」ボタンで、現在開いているタブの URL とタイトルをメモに紐付けられます。チップをクリックするとそのページを新しいタブで開けます。

- **選択テキストの取り込み**
  ページ上で文字列を選択して右クリック →「選択テキストを SideMemo に挿入」で、開いているメモのカーソル位置にそのまま挿入されます。

- **エクスポート / インポート**
  メモとタグを 1 つの JSON ファイルとしてダウンロードできます。マージ（既存に追加）と置き換え（既存削除して上書き）のどちらかを選んでインポートできます。

- **ローカル完結・外部送信ゼロ**
  すべてのメモは IndexedDB（あなたの端末）に保存されます。外部サーバーへの送信、テレメトリ、解析は一切行いません。

### 起動方法

- ツールバーの SideMemo アイコンをクリック
- キーボードショートカット `Ctrl+Shift+M`（Mac は `Cmd+Shift+M`）
- 任意のページで右クリック →「SideMemo を開く」

ショートカットは `chrome://extensions/shortcuts` から変更できます。

### 権限について

| 権限 | 用途 |
|------|------|
| `sidePanel` | サイドパネルとして動作するため |
| `storage` | 設定とセッション一時データ（選択テキスト挿入の中継）の保存 |
| `tabs` | 「ページ添付」で現在のタブの URL / タイトル / ファビコンを取得するため（ページ内容にはアクセスしません） |
| `contextMenus` | 右クリックメニュー項目の登録 |
| `downloads` | エクスポート JSON の保存 |

`host_permissions`（特定サイトへのアクセス権）は要求しません。

### 動作要件

- Chrome / Edge 114 以降（`chrome.sidePanel` API が必要です）

### プライバシー

データはすべてローカル（IndexedDB）に保存されます。サーバーへ送信したり、第三者と共有することはありません。詳しくはプライバシーポリシーをご覧ください。
